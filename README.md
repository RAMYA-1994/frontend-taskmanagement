## 🚀 **Features**
- **Task Creation**: Easily create and organize tasks.
- **Drag-and-Drop Interface**: Reorder tasks with a simple drag-and-drop feature.

- **Styling**: Tailwind CSS
- **State Management**: Redux Toolkit
- **APIs**: Axios for HTTP requests
